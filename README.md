# 🍺 Beer Counter

A lightweight PWA for tracking beers and splitting the bill. Works offline, installs on your home screen, and remembers everything across sessions.

## Features

- Count beers and track the running tab
- Set price per beer and currency
- Mark beers as paid — see outstanding balance at a glance
- Undo any action via toast notification
- Full history log
- Resets with a confirmation dialog (no accidental wipes)
- Installable as a PWA on iOS and Android
